### 2.

Таблицу можно выгрузить в файл и в последствии дополнить:

    $ bug_stat Aurata.dat > summary.csv
    $ bug_stat --append-to summary.csv Populii.dat
    ...

При этом Информация корректно дополняется - добавляются столбцы и строки.
